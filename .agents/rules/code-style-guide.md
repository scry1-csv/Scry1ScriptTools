# コーディング規約

## regionを用いた整理

C#の `#region` 機能を用いて、各クラス内のフィールド・プロパティ・メソッド等を整理する。
また、フィールド等をPublic・Privateなどのアクセスレベル毎でも分類する。
各 `#region`のラベルの例は以下。

- Constructor(s)
- Private Fields
- Public Methods
- XAML UI Event Handlers
- その他、あなたが分類が必要と思ったもの

## コメント方針

- 全体的な方針: コメントはできるだけ書かず、必要最小限にする。
- クラスやメソッド等の冒頭のドキュメントコメントは基本的に禁止する。名前・引数などのシグネチャのみでは機能の理解が難しい場合のみ、例外的にドキュメントコメントを許可する。
- メソッド内の処理の中では、ぱっと見で意図の理解が難しい部分がある場合のみ、コメントを許可する。
