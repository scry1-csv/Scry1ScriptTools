# Scry1ScriptTools

大人のオモチャ用の連動スクリプト用のツール集です。

- SexToyScriptViewer: スクリプトの内容をグラフ表示。一度に複数のスクリプトを同時に表示できる
- SexToyScriptConverter: スクリプトを他の形式へ変換する。変換前後の内容をグラフ表示できる
- 共通機能: 音声や動画ファイルを読み込んで再生することでチャート上に再生位置の目印を表示でき、音声を聞きながらスクリプトの確認が可能

## SexToyScriptViewer 対応スクリプト形式

- funscript
- U.F.O. SA/A10サイクロンSA用CSV
- U.F.O. TW用CSV
- タイムローター用CSV

A10ピストン用CSVは現在非対応です。(今後対応予定)

## SexToyScriptConverter 対応スクリプト形式

現在のところ、funscriptまたはUFOSA/A10サイクロンSA用CSVから、CoyoteScript（[WebSexToysPlayer](https://github.com/scry1-csv/WebSexToysPlayer)でCoyote V3を連動させるためのスクリプト形式）への変換のみに対応しています。
